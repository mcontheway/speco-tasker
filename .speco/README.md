# Speco-Tasker 配置目录

这个目录包含了Speco-Tasker项目的所有配置文件。

## 文件说明

### `config.json`
项目的基本配置信息，包括：
- 项目信息（名称、版本、描述等）
- 路径映射
- 功能开关
- 测试配置
- 代码质量工具配置
- 日志配置

### `paths.json`
路径配置和清理规则：
- 文件路径映射（重命名规则）
- 清理模式和排除规则

### `brand.json`
品牌重塑配置：
- 当前品牌信息
- 之前的品牌信息
- 重塑状态和变更记录

### `cleanup-rules.json`
清理规则定义：
- AI内容清理规则
- 文件重命名规则
- 配置迁移规则
- 备份和验证设置

## 子目录

### `logs/`
日志文件存储目录

### `backups/`
清理操作的备份文件

### `config/`
额外的配置文件（如环境特定配置）

## 注意事项

- 所有配置文件都应通过专用工具进行修改
- 不要直接编辑这些文件，除非您知道具体的格式要求
- 配置变更应通过相应的CLI命令或API接口进行
