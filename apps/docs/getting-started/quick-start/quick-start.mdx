This guide is for new users who want to start using Speco Tasker with minimal setup time.

It covers:
- [Requirements](/docs/getting-started/quick-start/requirements): You will need Node.js only.
- [Installation](/docs/getting-started/quick-start/installation): How to Install Speco Tasker.
- [Configuration](/docs/getting-started/quick-start/configuration-quick): Basic configuration options.
- [PRD](/docs/getting-started/quick-start/prd-quick): Writing and using PRDs for task planning.
- [Task Setup](/docs/getting-started/quick-start/tasks-quick): Preparing your tasks for execution.
- [Executing Tasks](/docs/getting-started/quick-start/execute-quick): Using Speco Tasker to execute tasks.
- [Rules & Context](/docs/getting-started/quick-start/rules-quick): Learn how to build context in your project over time.

<Tip>
By the end of this guide, you'll have everything you need to begin working productively with Speco Tasker.
</Tip>
