## Installation Options


<Accordion title="Option 1: MCP (Recommended)">

MCP (Model Control Protocol) lets you run Speco Tasker directly from your editor.

## 1. Add your MCP config at the following path depending on your editor

| Editor       | Scope   | Linux/macOS Path                      | Windows Path                                      | Key          |
| ------------ | ------- | ------------------------------------- | ------------------------------------------------- | ------------ |
| **Cursor**   | Global  | `~/.cursor/mcp.json`                  | `%USERPROFILE%\.cursor\mcp.json`                  | `mcpServers` |
|              | Project | `<project_folder>/.cursor/mcp.json`   | `<project_folder>\.cursor\mcp.json`               | `mcpServers` |
| **Windsurf** | Global  | `~/.codeium/windsurf/mcp_config.json` | `%USERPROFILE%\.codeium\windsurf\mcp_config.json` | `mcpServers` |
| **VS Code**  | Project | `<project_folder>/.vscode/mcp.json`   | `<project_folder>\.vscode\mcp.json`               | `servers`    |

## Manual Configuration

### Cursor & Windsurf (`mcpServers`)

```json
{
  "mcpServers": {
    "speco-tasker": {
      "command": "npx",
      "args": ["-y", "--package=speco-tasker", "speco-tasker"]
    }
  }
}
```

> **Note**: Speco Tasker requires no API keys or external configuration.

### VS Code (`servers` + `type`)

```json
{
  "servers": {
    "speco-tasker": {
      "command": "npx",
      "args": ["-y", "--package=speco-tasker", "speco-tasker"],
      "type": "stdio"
    }
  }
}
```

> **Note**: Speco Tasker requires no API keys or external configuration.

#### 2. (Cursor-only) Enable Speco Tasker MCP

Open Cursor Settings (Ctrl+Shift+J) ➡ Click on MCP tab on the left ➡ Enable speco-tasker with the toggle

#### 3. Initialize Speco Tasker

In your editor's AI chat pane, say:

```txt
Initialize speco-tasker in my project
```

</Accordion>

<Accordion title="Option 2: Using Command Line">

## CLI Installation

```bash
# Install globally
npm install -g speco-tasker

# OR install locally within your project
npm install speco-tasker
```

## Initialize a new project

```bash
# If installed globally
task-master init

# If installed locally
npx task-master init

# Initialize project with specific rules
task-master init --rules cursor,windsurf,vscode
```

This will prompt you for project details and set up a new project with the necessary files and structure.
</Accordion>
